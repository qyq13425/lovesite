关于求约会

发起约会时，你需要填写四个步骤:

  • 1 : 同意我们的协议 ;
  • 2 : 填写约会信息 ;
  • 3 : 填写对约会目标的要求 ;
  • 4 : 提交约会卡片 ;

当你的约会卡片提交后，你可以:

  • 1 : 在个人中心查看“我的约会” ;
  • 2 : 追加约会卡片上的信息 ;
  • 3 : 对应答卡片的人做出选择 ;
  • 4 : 回复别人对此次约会的评论 ;

当你选中心仪的目标，该约会卡片的状态 会变为“已配对成功”，其他人不能再提 交应答，但仍可以评论 ;

你可以在发布约会的选项中选择他人的评 论仅我可见，这样其他人就看不到别人对 该卡片做出的评论了 ;
